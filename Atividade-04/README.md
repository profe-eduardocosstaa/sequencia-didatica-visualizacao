Alinhada à Fase 4, esta atividade desafiou o raciocínio dedutivo através do jogo "Quem sou eu?". Grupos precisaram identificar sólidos a partir de descrições textuais ("cartões enigmáticos"), sem apoio visual. A dinâmica promoveu a negociação de significados entre pares para descartar hipóteses e convergir para a resposta correta. O registro na Ficha 4 exigiu a explicitação da justificativa geométrica para a escolha baseada nos atributos ouvidos.
